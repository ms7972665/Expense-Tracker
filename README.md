# 💰 ExpenseFlow — Smart Expense Tracker

ExpenseFlow is a **modern web-based expense tracker** designed to help users manage daily spending, analyze financial habits, and visualize expenses through interactive charts and analytics.

It features a **premium UI design, authentication system, real-time data management using Firebase, and beautiful financial insights dashboards**.

---

# 🚀 Features

### 🔐 Authentication

* Email & Password login
* Google Sign-in authentication
* Secure user session management using Firebase Auth

### 💵 Expense Management

* Add new expenses
* Edit existing expenses
* Delete transactions
* Category-based expense tracking

### 📊 Dashboard Analytics

* Total spending overview
* Average daily expense
* Top spending category
* Number of transactions

### 📈 Data Visualization

Interactive charts powered by **Chart.js**:

* Spending trend line chart
* Category distribution donut chart

### 🔍 Filters & Management

* Filter transactions by date
* Category filtering
* Sorting options
* Export data to CSV

### 🎨 Modern UI/UX

* Glassmorphism design system
* Dark / Light theme support
* Responsive layout
* Smooth animations
* Professional dashboard layout

# ⚙️ Firebase Setup

To enable authentication and database features:

1. Go to **Firebase Console**
2. Create a new project
3. Enable:

   * Authentication (Email/Password + Google)
   * Firestore Database
4. Copy your Firebase configuration into:

```
firebase-config.js
```

The repository includes a template configuration file.
